Alumn@: [Harry_Montenegro]

  1. Que es un sistema de control de versiones?

    [Es un software que nos permite el trabajo y colaboracion en equipo versionando los archivos en un punto en el tiempo con la finalidad de no afectar el trabajo de terceros]

  2. Cual es la diferencia entre git y github?

    [GitHub es un repositorio en la nube, mientras que git es el software de control de versiones]

  3. Cómo creo una rama en git?

    [git checkout -b ]

  4. Cómo clono un repositorio?

    [Si se va a clonar desde el repositorio en la nube se debe utilizar el comando git clone]
